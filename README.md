# 🌐 Static Website Hosting on AWS S3

## 📌 Project Overview
This project demonstrates how to host a static website using **Amazon S3**.  
It is a beginner-level AWS project to understand cloud storage and static website deployment.

---

## 🛠️ AWS Services Used
- Amazon S3 (Simple Storage Service)

---

## ⚙️ Project Steps
1. Created an S3 bucket
2. Uploaded `index.html` file
3. Enabled Static Website Hosting
4. Configured bucket policy for public access
5. Accessed website via S3 endpoint URL
